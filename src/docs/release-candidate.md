# Remaining Production Dependency

The only remaining production dependency is final photography integration.

Current placeholders intentionally preserve:

- layout
- spacing
- aspect ratios
- scroll geometry

When production assets are introduced they should:

- preserve the established aspect ratios
- include descriptive alt text
- use optimized formats (WebP or AVIF)
- use lazy loading where appropriate

No architectural changes are expected during this phase.

---

# Change Policy

Following RC1, all modifications should be evaluated as change requests.

Changes should preserve:

- Design Intent
- Story Freeze
- Motion Language
- Editorial Rhythm
- Motion Ownership architecture

Creative exploration should not resume without explicitly reopening the design phase.
