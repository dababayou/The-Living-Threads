# visual-narrative-freeze.md

Status: FROZEN

---

# Purpose

This document defines the visual language of the editorial journey.

It does not describe layouts.

It defines how the visitor's perception changes throughout the experience.

Story explains what changes.

Visual Narrative defines how that change is seen.

---

# Principle

Every chapter changes one aspect of perception.

The visitor should never feel they are entering a new webpage.

They should feel the camera is moving through one continuous film.

# Transition Principle

Transitions never reset the experience.

Each chapter inherits visual momentum from the previous chapter before establishing its own identity.

Transitions should feel like changes in attention, not scene cuts.

---

# Universal Principle

The editorial journey always follows:

Universal

↓

Specific

↓

Universal Again

The visitor leaves thinking less about Sumba alone,

and more about the universal human act of carrying culture forward.
